Quick Start

    # Cloning repository
        https://github.com/akshaydonhe12/Nimap-TEST


    Front End :
# Open directory
cd frontend

# Install dependencies for server
npm install

# Start server for frontend
npm start


Back End :
# Open directory
cd backend

# Install dependencies for server
npm install



# Start server for backend
npm start
